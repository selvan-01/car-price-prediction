# 🚗 Car Price Prediction using Random Forest

## 📌 Project Overview

This project predicts the **price of a car** based on various numerical features using a **Random Forest Regression model**. It also includes an **interactive Streamlit web application** for real-time predictions.

---

## 🎯 Objectives

* Build a machine learning model to predict car prices
* Understand regression using Random Forest
* Deploy the model using Streamlit with an attractive UI

---

## 🧠 Machine Learning Model

* Algorithm: Random Forest Regressor
* Type: Supervised Learning (Regression)
* Evaluation Metric: R² Score

---

## 📊 Dataset Information

* The dataset contains car features such as:

  * Engine size
  * Horsepower
  * Mileage
  * Other numerical attributes
* Target Variable: Price

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
